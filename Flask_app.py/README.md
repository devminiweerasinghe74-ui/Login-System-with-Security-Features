# 🔐 Login System with Security Features

A secure authentication web application built using Flask and SQLite with password hashing, login protection, session management, and modern UI.

---

## 🚀 Features

- User Registration & Login
- Password Hashing
- Email Validation
- Flask Sessions
- Account Locking
- SQLite Database
- Responsive UI Design
- Secure Authentication System

---

## 🛠 Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- Jinja2
- Gunicorn

## 🔐 Security Features

### Password Hashing
Passwords are securely hashed before storing in the database.

### Account Locking
Accounts are locked after multiple failed login attempts.

### Session Authentication
Flask sessions are used to maintain logged-in users securely.
